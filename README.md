# ENM Workflows 2024
These workflows were created as part of a collaborative project. Once the project is complete, I will link to the paper. Data for each workflow are downloaded as part of the script. Thus, each script is self-contained and demonstrates how to document the code using the [rangeModelMetadata](https://cran.r-project.org/web/packages/rangeModelMetadata/vignettes/rmm_vignette.html) package. I have also included [ODMAP](https://odmap.wsl.ch/) protocols for each.
